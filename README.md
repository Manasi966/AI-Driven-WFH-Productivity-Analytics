AI-Driven Work-From-Home Productivity & Behavioral Analytics Platform

This project analyzes work-from-home focus behavior using computer vision
and visualizes productivity analytics through a secure web dashboard.

Tech Stack:
- Python
- OpenCV
- Pandas
- Matplotlib
- Streamlit

Run locally:
pip install -r requirements.txt
python main.py
python analytics.py
python -m streamlit run dashboard.py
